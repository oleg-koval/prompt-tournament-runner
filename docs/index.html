Markdown-first refactor runner

AI Refactor Playbook Runner

Repeatable markdown refactors with inheritance, placeholder expansion, retries, timeout reporting, dry-run, and resume. Built for CLI use and a local browser UI you can open from mobile over Tailscale.

Read the README GitHub repo Releases Issues

Quick start

npm ci
npm run build
node dist/cli.js run examples/demo.md --dry-run
node dist/cli.js serve --host 0.0.0.0 --port 8787

From mobile, open the server on your Tailscale IP or MagicDNS hostname.

What it does

  • Parses reusable playbooks from markdown
  • Merges inherited templates without losing overrides
  • Interpolates {{placeholders}} in steps and commands
  • Runs sequentially with retries and per-step timeouts

Use it for

  • domain rename sweeps
  • cross-repo cleanup passes
  • repeatable convention rollouts
  • safe dry-run planning before the agent touches files
